Palindrome Detector from Marsel Sharapov

  Description

A C# program for detecting palindromes.
The project consists of the following files:

Program.cs — the main program file.

Detector.cs — contains the palindrome detection logic.

PalindromeDetectorFromMarsel.csproj — the C# project file.

PalindromeDetectorFromMarsel.sln — solution file for Rider/Visual Studio.

  How to Run

Download the repository from GitHub or use the Code → Download ZIP button.

Extract the ZIP file to your computer.

Open the MarsPlaindrome-master folder that you extracted in Rider (or Visual Studio).
  
Open Terminal on the left side of interface.

Run the project by pressing Run.

The program is ready to use.

Write some words to check it.

  Notes

Make sure you have a .NET SDK installed that is compatible with the project.

All necessary files are included in the repository.
